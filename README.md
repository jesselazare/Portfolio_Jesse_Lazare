Procédure & Informations :  - Clonage du dépôt Ouvrez un terminal et récupérez le code source depuis GitHub 
             - Installation des dépendances Installez tous les paquets nécessaires (Next.js, React, Resend, etc.) listés dans le package.json :
             - Configuration des variables d'environnement Le projet nécessite une clé API pour le service d'envoi d'emails.
             - Créez un fichier nommé .env.local à la racine du projet.
             - Ajoutez-y votre clé API Resend comme ceci : RESEND_API_KEY=re_votre_cle_secrete_ici
             - Lancement du serveur de développement Démarrez le projet en mode local : npm run dev
             Accès au site Une fois le serveur lancé, ouvrez votre navigateur et rendez-vous à l'adresse suivante : http://localhost:3000
